# Contributing to Galaxy Guard Ohio

We welcome suggestions, bug reports, and contributions from the community!

## How to Contribute

1. **Fork the repository** and create your branch.
2. **Open an Issue** for bugs, feature requests, or questions.
3. **Submit a Pull Request** with a clear description and reference to any related issues.
4. **Discuss in GitHub Discussions** for ideas, help, or feedback.

## Code Style
- Use Prettier and ESLint for formatting.
- Write clear, accessible React components.
- Add tests for new features.

## Communication
- Use Issues for bugs and feature requests.
- Use Discussions for help, suggestions, and collaboration.

## Getting Help
- Tag your issue with `help wanted` or `question`.
- Join our GitHub Discussions for community support.
